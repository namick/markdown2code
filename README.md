# Markdown2code

Converts Markdown into commented executable code

## Installation

Add this line to your application's Gemfile:

    gem 'markdown2code'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install markdown2code

## Usage

    require 'markdown2code'

    Markdown2code.render(markdown_text)
